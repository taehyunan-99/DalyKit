# Stat 보고서 가이드

## 데이터 소스

- 입력: `dalykit/kits/{kit}/stat/stat_results.json`
- 출력: `dalykit/kits/{kit}/stat/stat_report.md`
- 이미지: `dalykit/kits/{kit}/stat/figures/`

## 필수 섹션

1. 요약
2. 연구 개요
3. 전제 조건 확인
4. 가설 수립 및 검증
5. 사후 분석 결과
6. 실질적 해석 및 인사이트
7. 한계점 및 추가 분석 추천

## 이미지 참조 규칙

예:

```markdown
![H1 검증 시각화](./figures/stat_h1_boxplot.png)
```
