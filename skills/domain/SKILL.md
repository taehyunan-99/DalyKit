# Domain (도메인 파일 구조화)

> `domain.md`의 자유 입력 섹션과 `data/raw/` 데이터를 읽어 구조화 정보를 갱신한다.

## 사용법

```text
dalykit:domain
```

## 사전 조건

- `dalykit/config/domain.md`가 있어야 한다
- `dalykit/data/raw/`에 CSV 또는 Excel 파일이 있으면 함께 참조한다
- 없으면 `dalykit:init`을 먼저 실행하라고 안내한다

## 워크플로우

### 1단계: 입력 수집

1. `dalykit/config/domain.md`에서 `# 자유 입력` 섹션을 읽는다
2. `dalykit/data/raw/`를 스캔해 CSV/Excel 파일을 찾는다
3. 파일이 1개면 자동 선택, 여러 개면 선택을 요청한다
4. 선택한 데이터에서 컬럼명, dtype, nunique, 기본 통계를 수집한다

### 2단계: 구조화

자유 입력과 데이터 프로파일을 조합해 아래를 채운다.

- 업종
- 데이터 설명
- 핵심 타겟 변수
- 주요 피처
- 컬럼 설명
- 도메인 규칙
- 비즈니스 제약
- 분석 제외 컬럼
- 특이사항
- ML 목표

### 3단계: 파일 갱신

- `# 자유 입력` 섹션은 보존한다
- `---` 아래 `# 구조화 정보` 섹션만 갱신한다
- 추론한 항목은 `(추론)` 태그를 붙인다

## 규칙

1. 자유 입력 섹션은 수정하지 않는다
2. 원본 데이터 경로는 항상 `dalykit/data/raw/` 기준으로 본다
3. 불확실한 항목은 "미지정 — 직접 입력하세요"로 표기한다
4. 모든 설명은 한국어로 작성한다
